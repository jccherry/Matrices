#Matrices

Class to create and edit 2 dimensional Matrices

Includes functions to convert the Matrix to REF and RREF

Sample output:
```
[1.0, 2.0, 2.0, 4.0]
[1.0, 3.0, 3.0, 5.0]
[2.0, 6.0, 5.0, 6.0]

Swap R1 & R1

[1.0, 2.0, 2.0, 4.0]
[1.0, 3.0, 3.0, 5.0]
[2.0, 6.0, 5.0, 6.0]

R2 - 1.0R1 -> R2

[1.0, 2.0, 2.0, 4.0]
[0.0, 1.0, 1.0, 1.0]
[2.0, 6.0, 5.0, 6.0]

R3 - 2.0R1 -> R3

[1.0, 2.0, 2.0, 4.0]
[0.0, 1.0, 1.0, 1.0]
[0.0, 2.0, 1.0, -2.0]

Swap R2 & R2

[1.0, 2.0, 2.0, 4.0]
[0.0, 1.0, 1.0, 1.0]
[0.0, 2.0, 1.0, -2.0]

R3 - 2.0R2 -> R3

[1.0, 2.0, 2.0, 4.0]
[0.0, 1.0, 1.0, 1.0]
[0.0, 0.0, -1.0, -4.0]

1/-1.0R3 -> R3

[1.0, 2.0, 2.0, 4.0]
[0.0, 1.0, 1.0, 1.0]
[-0.0, -0.0, 1.0, 4.0]

R2 - 1.0R3 -> R2

[1.0, 2.0, 2.0, 4.0]
[0.0, 1.0, 0.0, -3.0]
